#### 阅读方法
本书共分为十章，每一章都是一个专题，专门讲解一类算法问题，每一章中又划分多个小节，专门讲解这类算法中的某一分支。同一章节中的算法会有明显的联系，我把较为简单的算法放在前面，复杂的算法放在后面，很多复杂的算法会使用之前的基础算法，本书尽量不重复工作，而是尽量直接调用。每个算法都会附上测试用例。
