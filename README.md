#### 阅读方法
本书的每一章专门讲解一类算法问题，其中又划分多个小节，专门讲解某一分支或变种问题。
同一章节中各个小节之间会有明显的联系，简单的算法在前面，复杂的算法在后面。
每个算法都有讲解、源码和测试三个部分。
